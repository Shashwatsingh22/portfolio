# Web Resume Project

## Overview
This is a static HTML/CSS/JS resume site for Shashwat Singh. All resume data is driven from `data.json` — edit that file to update content. The page rebuilds automatically.

## Structure
- `index.html` — Entry point, loads style.css and app.js
- `style.css` — Clean, print-friendly, mobile-responsive styling
- `app.js` — Reads data.json via fetch and renders the resume
- `data.json` — All resume data (edit this to update)

## Key Rules
- All content changes should go in `data.json` only
- HTML structure is generated by `app.js` — don't hardcode content in `index.html`
- CSS uses a max-width of 850px centered layout
- Print styles are included for clean PDF export via browser print
- Mobile responsive at 600px breakpoint

## data.json Structure
- `name`, `email`, `phone`, `links[]` — Header/contact info
- `summary` — Summary paragraph
- `experience[]` — Each entry has: company, note (optional), location, role, duration, points[], subSection (optional with heading + points), extraPoints[] (optional)
- `education[]` — institution, location, degree, duration
- `achievements[]` — title, description
- `skills{}` — key-value pairs (category: comma-separated values)

## Hosting
- Works on GitHub Pages, Netlify, Vercel, or any static host
- For local testing: `python3 -m http.server 8080` in the web-resume directory
